# Job-Searcher
A job searcher coded in Python to scrape the internet for real time job listings using custom search queries and locations. Uses the SerpAPI Google Jobs engine.

## 1. Prerequisites
* **Python:** Download and install from [python.org](https://www.python.org/downloads/).
* **API Key:** Create a free account at [serpapi.com](https://serpapi.com/) and copy your Private API Key from the dashboard, and insert it into Line 35 of the code.
